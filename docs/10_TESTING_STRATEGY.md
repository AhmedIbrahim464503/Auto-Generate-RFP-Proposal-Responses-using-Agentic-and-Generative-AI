# 10. Testing Strategy

## Overview
A comprehensive strategy spanning unit tests, integration tests, and LLM output evaluation.

## Unit Testing
- Test parsers on dummy PDF and DOCX files.
- Test retriever vector search relevance using NDCG.

## LLM Evaluation
- Apply Ragas framework (faithfulness, answer relevance, context recall).
- Run regression tests on standard RFP benchmarks.
