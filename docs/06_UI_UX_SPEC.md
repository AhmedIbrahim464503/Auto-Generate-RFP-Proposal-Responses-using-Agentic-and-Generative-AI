# 06. UI/UX Specifications

## Core Pages
1. **Dashboard**: High-level view of ongoing RFPs, compliance rates, and completion progress.
2. **Project Upload**: Drag-and-drop workspace to ingest RFPs and background knowledge.
3. **Workspace (Split Screen)**:
   - *Left*: Parsed RFP questions.
   - *Right*: AI-generated response, draft editing panel, and reference citations.
4. **Knowledge Base Manager**: Admin dashboard to manage and update source documents.
