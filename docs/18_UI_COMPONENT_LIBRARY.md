# 18. UI Component Library

## Design System Tokens
- Colors: Curated Slate, Emerald, Indigo gradients.
- Typography: Inter, Outfit.

## Key Shared Components
- `Button`: Sleek hover states and loading spin indicators.
- `Card`: Container with subtle hover shadow/border effects.
- `SplitPane`: Vertical/Horizontal resizable splits.
- `DocViewer`: Sidebar displaying PDF/DOCX content with highlighting.
